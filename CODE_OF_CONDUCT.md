# Code of Conduct

ScieMD is intended to support open scientific software collaboration. We expect everyone
participating in this project to treat others with respect, patience, and professionalism.

## Expected Behavior

- Use welcoming and inclusive language.
- Respect different scientific backgrounds, skill levels, workflows, and viewpoints.
- Give and receive constructive feedback in good faith.
- Focus criticism on ideas, code, documentation, and evidence rather than individuals.
- Respect privacy, confidentiality, and research data handling requirements.

## Unacceptable Behavior

- Harassment, intimidation, personal attacks, or sustained disruption.
- Discriminatory comments or conduct.
- Publishing private information without permission.
- Misrepresenting results, authorship, affiliations, or security findings.
- Any behavior that project maintainers reasonably determine is inappropriate for a
  professional open source scientific community.

## Reporting

Report code of conduct concerns privately to `info@scienfy.com`. Include relevant links,
screenshots, dates, and context when possible. Scienfy Inc. will review reports privately
and take action when needed, which may include warnings, moderation, temporary limits, or
removal from project spaces.

This code of conduct is adapted for ScieMD and informed by the Contributor Covenant
approach to open source community standards.
